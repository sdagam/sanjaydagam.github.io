Least Squares and Pseudoinverses: Solving Unsolvable Systems

#126 of 156 May 30, 2026 Group 5 · Jan–Jun 2026

In 2009, the FBI offered a $1 million reward for information about the most wanted criminals. Investigators had surveillance footage — blurry, pixelated images — and needed to reconstruct clearer photographs. The mathematical problem: they had many low-quality measurements (blurry pixels) and needed to find the high-resolution image most consistent with all of them. No single exact answer existed — the system was overdetermined, with more equations than unknowns. The tool that finds the best possible answer to an unsolvable system is the pseudoinverse, and it's one of the most broadly used results in all of applied mathematics.

When Exact Solutions Don't Exist

You've probably solved systems of equations where the number of equations equals the number of unknowns — and if everything works out, there's exactly one solution. But real problems often have more equations than unknowns (overdetermined) or more unknowns than equations (underdetermined).

Overdetermined: more measurements than unknowns. A GPS receiver gets signals from 8 satellites, each giving one equation relating your position to the satellite's position. But you only have 3 unknown coordinates. The 8 equations are inconsistent (due to measurement noise) — no exact solution exists. Which approximate solution is best?

Underdetermined: more unknowns than equations. A compressed sensing problem might recover an image from far fewer measurements than pixels — using the additional knowledge that real images are sparse in some basis. Infinitely many solutions satisfy the equations — which one should you choose?

Least Squares: The Best Approximate Solution

For overdetermined systems Ax = b where no exact x exists, the least squares solution minimizes ‖Ax - b‖² — the sum of squared differences between the left and right sides. It's the x that makes the equations "as true as possible" in total.

Least squares: minimize ‖Ax - b‖² Normal equations solution: x̂ = (AᵀA)⁻¹Aᵀb Aᵀ = transpose of A (rows become columns) AᵀA = a square matrix (always) (AᵀA)⁻¹ = inverse of AᵀA In plain English: the formula (AᵀA)⁻¹Aᵀ extracts the best possible x from an inconsistent system. It's exact when a solution exists and minimizes total squared error when it doesn't.

The Pseudoinverse: One Formula for All Cases

The expression (AᵀA)⁻¹Aᵀ is called the Moore-Penrose pseudoinverse of A, written A⁺. It's defined even when A is not square and even when AᵀA is singular (not invertible). The pseudoinverse uses the SVD (Article #91) to handle all cases uniformly:

Pseudoinverse via SVD: A = UΣVᵀ (SVD decomposition) A⁺ = VΣ⁺Uᵀ Σ⁺ = replace each nonzero diagonal entry σᵢ with 1/σᵢ leave zero entries as zero (don't invert them) For overdetermined (more equations than unknowns): A⁺b = least squares solution (minimizes ‖Ax - b‖²) For underdetermined (more unknowns than equations): A⁺b = minimum norm solution (smallest ‖x‖ among all exact solutions)

The pseudoinverse automatically handles both cases. For overdetermined systems, it gives the least squares solution — best approximate answer. For underdetermined systems, it gives the minimum-norm solution — the most conservative choice among all exact solutions. One formula, all cases.

Image Reconstruction

For the FBI's blurry surveillance images: each blurry pixel is a weighted average of nearby sharp pixels — a linear equation. Many blurry pixels give many equations relating the unknown sharp pixel values. The system is overdetermined and inconsistent (due to sensor noise). The pseudoinverse finds the sharp image that best explains all the blurry observations simultaneously — the least squares reconstruction. Modern image deblurring software uses exactly this approach, extended with regularization to prevent noise amplification.

Other Applications

Pseudoinverses solve GPS positioning (overdetermined from many satellites), fit polynomial curves to data points (least squares curve fitting), control robot arms (underdetermined — many joint configurations reach a target), and reconstruct signals in MRI and CT imaging. Any time you need the best answer to a system that doesn't have an exact one, the pseudoinverse provides it.

Conclusion

Real problems rarely come with systems of equations that have exactly one solution. The least squares solution handles overdetermined systems by minimizing total squared error. The pseudoinverse handles all cases — overdetermined and underdetermined — with one unified formula derived from SVD. For GPS positioning, image reconstruction, robot control, and medical imaging, the pseudoinverse provides the best possible answer when no perfect answer exists. It's the mathematical definition of "good enough" — proven to be the best you can do.
